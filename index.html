🗺️ Sobre esta aplicación

Esta aplicación muestra todas las actividades que el Ayuntamiento de Madrid organiza en sus diferentes centros en los próximos 100 días. Los datos se cargan en tiempo real utilizando la API abierta del Ayuntamiento.

Se han realizado ajustes para agrupar la información por tipo de evento de manera más comprensible.

🔗 Fuente de datos

  • Portal de datos abiertos del Ayuntamiento de Madrid
  • API de eventos culturales (próximos 100 días)

⚙️ Tecnologías utilizadas

Leaflet.js Leaflet.markercluster Chart.js Inter (Google Fonts) HTML/CSS/JS Vanilla CartoDB Positron tiles
